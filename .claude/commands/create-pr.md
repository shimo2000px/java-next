GitHubにPull Requestを作成し、AIコードレビューをPRコメントとして投稿します。

## 手順

1. 現在のブランチと main との差分を確認する:
   ```
   git log main...HEAD --oneline
   git diff main...HEAD
   ```

2. 以下のテンプレートでPRを作成する:
   ```
   gh pr create --title "<タイトル>" --body "$(cat <<'EOF'
   ## 概要
   <!-- 変更内容を簡潔に -->

   ## 変更の種類
   - [ ] バグ修正
   - [ ] 新機能
   - [ ] リファクタリング
   - [ ] ドキュメント
   - [ ] その他

   ## 変更内容
   <!-- 具体的な変更点を箇条書きで -->

   ## テスト
   - [ ] 既存テスト通過
   - [ ] 新規テスト追加

   ## 確認事項
   - [ ] セキュリティ上の問題なし
   - [ ] 破壊的変更なし

   🤖 Generated with Claude Code
   EOF
   )"
   ```

3. PRを作成したら、すぐに `/ai-review` スキルと同等のAIレビューを実行し、結果を `gh pr comment <PR番号> --body "..."` でコメントとして投稿する。

4. PRのURLとレビューコメントのURLを表示する。

## 注意
- `$ARGUMENTS` があればタイトルとして使用する
- リモートへのpushが必要な場合は事前にユーザーに確認する
